# Documentation

- `reference/` — API spec, errors, local dev, known sharp edges
- `conventions/` — coding and testing standards
- `iterations/` — [README](iterations/README.md): current iteration scope under `iterations/current/`, plus evergreen [change routing](iterations/change-routing.md)
