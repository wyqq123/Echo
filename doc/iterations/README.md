# Iterations

This folder holds **iteration planning** and **evergreen change guidance** for the repo.

| Path | Role |
|------|------|
| [current/README.md](current/README.md) | **Current iteration**: active goals, in-scope work, blockers, release notes, and deferred items for this cycle. |
| [change-routing.md](change-routing.md) | **Change routing** (not sprint-specific): where to start editing and what to verify by product area (Funnel, Timeline, persistence, auth, etc.). |

Agents should read `current/` when prioritizing work for the active iteration, and `change-routing.md` when routing a concrete code change to the right files and layers.
